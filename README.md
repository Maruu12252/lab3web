# My SPA на Vite + Vue 3

## Опис
Простий SPA з трьома сторінками:
- **Home** — короткий опис застосунку.
- **Catalog/List** — список елементів зі статичного JSON або константи.
- **Details** — детальний перегляд елемента за маршрутом `/items/:id`.

Навігація між сторінками без перезавантаження (Vue Router).

## Запуск
1. Встановити залежності:
   npm install
2. Запуск локального сервера для розробки:
   npm run dev
3. Створення збірки:
   npm run build
4. Перевірка збірки:
    npm run preview
